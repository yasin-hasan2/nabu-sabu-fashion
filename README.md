A modern full-stack MERN e-commerce web application for showcasing fashion products and collecting user reviews with secure authentication.

---

## 🚀 Live Demo

- 🌐 Frontend: https://nabu-sabu-fashion.vercel.app  
- ⚙️ Backend: https://nabu-sabu-fashion.onrender.com  

---

## ✨ Features

- 🔐 Authentication (Register / Login / Logout)
- 🍪 Secure Cookie-based JWT Auth
- 👤 User Profile System
- 🛍️ Product Listing & Details
- ⭐ Review & Rating System
- 🔒 Protected Routes
- ⚡ Fast UI with Vite + React
- 📱 Fully Responsive Design

---

## 🧰 Tech Stack

### 🎨 Frontend
- React (Vite)
- Tailwind CSS
- Axios

### ⚙️ Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

⚙️ Environment Variables
📦 Backend (server/.env)

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=https://nabu-sabu-fashion.vercel.app

🌐 Frontend (client/.env)
VITE_API_URL=https://nabu-sabu-fashion.onrender.com

🧪 Run Locally
1️⃣ Clone Repository
git clone https://github.com/yasin-hasan2/nabu-sabu-fashion.git
cd nabu-sabu-fashion

2️⃣ Backend Setup
cd server
npm install
npm run dev

3️⃣ Frontend Setup
cd client
npm install
npm run dev

🚀 Deployment
Frontend: Vercel
Backend: Render
Database: MongoDB Atlas

⚠️ Important Notes
⏱️ Backend may sleep (Render free tier)
🐢 First request can be slow
🍪 Cookies require:
secure: true
sameSite: "none"
